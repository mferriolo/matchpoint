Crelate-Bridge Settings

v0.1

Server-side configuration

The Crelate API key lives in Supabase secrets, not the extension. To rotate it, update CRELATE_API_KEY in the Supabase dashboard (Project Settings → Edge Function Secrets) and redeploy the extension-bridge function.

Conflict policy

Always-ask (path C). Any field-level divergence opens a side-by-side picker. There's no auto-merge — you click the winner per field.
